# ShopSphere

A mini E-Commerce web application built using Java JSP, Servlets, JDBC, MySQL, HTML, and CSS.

## Features

* User Registration
* User Login & Logout
* Session Management
* Product Listing
* Add To Cart
* View Cart
* Remove From Cart
* MySQL Database Integration

## Technologies Used

* Java
* JSP
* Servlets
* JDBC
* MySQL
* HTML
* CSS
* Apache Tomcat

## Database

Create a MySQL database named `ecommerce`.

Required tables:

* users
* products
* cart

## How to Run

1. Import project into IntelliJ IDEA.
2. Configure MySQL database.
3. Update database credentials in `DBConnection.java`.
4. Deploy on Apache Tomcat.
5. Open:

http://localhost:8080/ShopSphere
